Ce projet réalisé en groupe consistait à réaliser un programme parfaitement fonctionnel permettant de charger, éditer et sauvegarder un fichier de données structurées. Ce programme devait disposer d'une interface graphique et devait être développé de façon multi-plateforme.

  • Chargement des données et sauvegarde au format JSON/XML
  • Affichage & édition via interface graphique (Présentation en mode master/detail)
  • Filtrage dynamique des données et Internationalisation
  • Application Multi-plateforme (Compatible Windows et Linux)

    Ce projet m’a permis de consolider et développer plusieurs compétences en :
  • Programmation en C++ moderne
  • Utilisation avancée de Qt (Qt Creator, QML, Qt Linguist)
  • Conception d’IHM ergonomiques et dynamiques (responsive design)
  • Manipulation de formats de données structurés (JSON, XML)
  • Utilisation de Git & GitHub pour le travail collaboratif
  • Travail en binôme : répartition équitable des tâches, gestion de projet
  • Création d’un diagramme UML (classes, interactions)

Outils de développement :

Frameworks/Bibliothèques :

  • Qt 6, Qt Linguist, Qt JSON/XML
  • QRegularExpression

  • Gestion des expressions régulières pour le filtrage
  • Synchronisation collaborative avec Git (gestion des conflits)
  • Affichage conditionnel des sous-formulaires dynamiques

  • Utilisation d’une bibliothèque de regex robuste intégrée à Qt
  • Mise en place d’une convention Git stricte : commits fréquents, branches dédiées, rebase local avant push
  • Refactorisation de l’IHM en composants modulaires : chaque bloc gérant un sous-formulaire était indépendant, facilitant l’édition
